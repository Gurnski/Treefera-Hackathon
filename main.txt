  What you've got

  A dark, glassmorphic single-page satellite-intelligence dashboard. All seven
  sections are wired to static data, no backend or live API calls:

  1. GlobeHero — hand-built SVG/CSS orbital globe with animated radar sweep,
  pulsing site marker, and a "TARGET LOCK" coordinate readout. I deliberately
  skipped react-globe.gl to avoid WebGL/dependency risk in a demo; the fallback
  reads like the real thing and has zero extra deps.
  2. RegenScoreCard — animated radial gauge (72/100, "Medium-high signal"),
  factor chips with weight bars, and a "Signal, not proof" badge.
  3. TargetVsControlChart — Recharts composed chart, target NDVI area vs control
  dashed line, with the suspected adoption window shaded. Empty-state if the
  timeseries is missing.
  4. EvidencePanel — before/after tiles. If a PNG fails to load it swaps to a
  CSS placeholder (no broken-image icon), captions note real exports can replace
  them.
  5. AiInsightCard — prewritten narrative from manifest.json, no OpenAI call,
  with an honest disclaimer.
  6. LimitationsCard — confidence meter + the five scientific caveats.
  7. ImpactPanel — the triage thesis and concrete validation path.

  Data contract (the swap-tomorrow part)

  - public/demo-data/manifest.json, ndvi_timeseries.json, before.png, after.png
  - src/types.ts mirrors the JSON exactly; src/lib/loadDemoData.ts is the single
  fetch point and reads the timeseries path from the manifest.
  - Nothing is hardcoded in components — they take typed props from the loaded
  data.
  - To go live: overwrite those four files with notebook exports matching
  types.ts. Comments in the JSON and types call this out.
